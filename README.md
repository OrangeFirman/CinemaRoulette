# CinemaRoulette

![cr](https://github.com/user-attachments/assets/b591b6cc-2be3-4ae9-ad99-5439bf2d5160)

Локальный инструмент для просмотра фильмов без страховки в виде рейтингов, трейлеров и рекомендаций. Три режима, один принцип: соглашайся, не зная, на что идёшь.

Работает на [Kinopoisk Unofficial API](https://kinopoiskapiunofficial.tech). Запускается полностью локально — никаких аккаунтов, слежки и облаков.

---

## Режимы

| Режим | Описание |
|-------|----------|
| 🎭 **Тайный Фильм** | Один случайный фильм. Без постера, названия и контекста. Узнаёшь, что смотришь, только когда начнётся. |
| 🎰 **Кино-Рулетка** | Шесть фильмов в барабане. Пять хороших, один провальный. Постеры размыты. Крутишь или смотришь — но проверить заранее нельзя. |
| 🎬 **1 из 3** | Три случайных фильма — только постер и название. Достаточно, чтобы выбрать, недостаточно, чтобы судить. |

## Установка

**Требования:** Python 3.8+ и бесплатный ключ [Kinopoisk Unofficial API](https://kinopoiskapiunofficial.tech).

**Просто** — скачай `.exe` ниже и запусти двойным кликом. Установка не нужна.

**По-человечески** — запусти `CinemaRoulette.bat` или:

```bash
python cinaroulette.py
```

Затем открой **http://localhost:7777** в браузере.

При первом запуске нажми **API KEY** в правом верхнем углу и вставь ключ Кинопоиска. Он сохранится в `api_key.json` рядом со скриптом и будет загружаться автоматически при следующих запусках.

---

## Опционально: постеры через TMDb

CDN Кинопоиска бывает нестабильным. Для надёжной загрузки постеров добавь бесплатный ключ [TMDb API](https://www.themoviedb.org/settings/api) (v3 auth) в том же окне **API KEY**. Если постер с Кинопоиска не загружается, приложение автоматически запросит его с TMDb. Также исправляет несовпадение языка постеров при переключении на английский интерфейс.

Ключ TMDb хранится в том же файле `api_key.json`.

## Зачем это существует

Каждый фильм приходит к тебе уже с готовым вердиктом. Рейтинги, трейлеры, рецензии — к моменту, как ты его включаешь, ты уже знаешь, что думать. Смотришь вслепую — и что-то меняется. Ты будешь не соглашаться с консенсусом чаще, чем ожидаешь. В этом и смысл.

Квентин Тарантино развил свой вкус не на великих фильмах. Он смотрел всё подряд, включая плохое. Плохие фильмы учат тому, чему хорошие научить не могут — когда что-то не работает, механика видна. Замечаешь швы.

CinemaRoulette создан не для тех, кто хочет смотреть лучшее. Он для тех, кто хочет по-настоящему развить вкус.

## Использование API

Бесплатный тариф Kinopoisk Unofficial API позволяет делать **500 запросов в день** — счётчик виден в шапке и сбрасывается в полночь. Каждый запуск режима стоит примерно 3–15 запросов в зависимости от фильтров. Рулетка самая дорогая (~12 запросов на пять хороших фильмов плюс несколько на провальный).
